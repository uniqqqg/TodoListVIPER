Особенности проекта

- Архитектура VIPER — чёткое разделение ответственности по слоям (View, Interactor, Presenter, Entity, Router)
- UIKit - пользовательский интерфейс построен с использованием UIKit без Storyboard
- Core Data — локальное хранение задач с корректным восстановлением при повторном запуске
- Загрузка данных из API — при первом запуске данные загружаются с dummyjson.com и сохраняются в Core Data
- GCD (Grand Central Dispatch) — все операции с сетью и базой данных выполняются в фоновом потоке
- Контекстное меню — долгое нажатие на задачу открывает меню редактирования и удаления
- Поиск задач — фильтрация списка по названию в реальном времени
- Современный Swift — использование последних возможностей языка и принципы чистой архитектуры
- 
 Функционал

- Просмотр списка задач
- Поиск задач по названию
- Изменение статуса задачи (выполнена/не выполнена)
- Добавление новых задач
- Редактирование существующих задач
- Удаление задач
- Сохранение всех изменений в Core Data

Стек технологий

- Swift
- UIKit
- Core Data
- URLSession + GCD
- VIPER Architecture
